# 跨境电商智能答疑小助手

## 项目简介
本项目是一个面向TikTok Shop跨境电商学员的智能问答系统，通过意图识别、知识库检索与大模型生成相结合的方式，自动回答学员在学习和运营过程中遇到的常见问题。

## 核心功能
- **意图识别**：基于固定短语匹配，识别问候、感谢、告别、人工服务等意图，给出对应回复。
- **知识库问答（RAG）**：基于300+条跨境电商常见问答对，精准回答设备准备、网络环境、账号注册、选品上架、供应链、支付提现等问题。
- **大模型自回归**：当知识库信息不足时，结合大模型自身知识进行补充回答。
- **人工服务转接**：识别用户转人工需求，提供人工答疑入口。

## 技术栈
- Python（意图识别模块）
- 大语言模型（RAG + 自回归）
- 知识库：Excel问答对

## 使用场景
- 跨境电商课程学员24小时自助答疑
- 降低导师重复性问答负担
- 覆盖从手机刷机到店铺运营全流程

## 文件说明
- `1.py`：固定短语匹配的意图识别模块
- `LLM系统提示词(RAG).md`：基于知识库检索的回答提示词
- `LLM系统提示词(基于大模型自身).md`：基于大模型自身知识的回答提示词
- `数据.xlsx`：跨境电商常见问题与答案（300+条）
